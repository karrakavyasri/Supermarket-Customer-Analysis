# 🛒 Supermarket Customer Analysis

## 📌 Project Overview
This project analyzes supermarket customer data using Python to understand customer behavior, identify purchasing patterns, and visualize product combinations.  
The insights from this analysis can help supermarkets improve marketing strategies, product placement, and customer retention.

---

## 🎯 Objective
- Segment customers based on income and spending behavior  
- Analyze buying patterns of products  
- Identify product combinations frequently purchased together  

---

## 🧰 Tools & Technologies Used
- **Python**
- **Pandas** – data manipulation
- **Matplotlib** – data visualization
- **Scikit-learn** – customer segmentation (K-Means clustering)

---

## ✨ Features
- Customer segmentation using **K-Means Clustering**
- Product-wise buying pattern analysis
- Product combination visualization using correlation heatmap
- Single-page dashboard with **3 visualizations**

---

## 📊 Output Visualizations
1. **Customer Segmentation Graph** – groups customers based on income and spending score  
2. **Product Buying Pattern Graph** – shows most frequently purchased products  
3. **Product Combination Graph** – visualizes relationships between products bought together
